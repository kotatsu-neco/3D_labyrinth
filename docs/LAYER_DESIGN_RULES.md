# LAYER_DESIGN_RULES

この文書は、地下方舟型迷宮の階層設計ルールを記録する正本候補である。
`docs/STORY_CORE.md` の物語核を前提とする。

## 1. 基本方針

- 迷宮は、災厄から生命・知識・財貨・血統を保存するために作られた地下方舟である。
- 構造は固定マップを前提とし、毎回増殖・変化する迷宮にはしない。
- 階層が深くなるほど、方舟の中枢・管理区・神殿・書庫・研究区・高位保管区に近づく。
- 浅い階層ほど、外縁部・搬入口・倉庫・飼育区・廃棄区に近い。
- 敵、宝、イベント、施設の価値と性質は、この階層勾配に従う。

## 2. 階層勾配

階層設計では、深さに応じて以下の変化を持たせる。

- 生物: 原始的・野生的なものから、知性的・社会的・魔術的なものへ。
- 社会: 群れ、巣、縄張りから、部族、信仰、階級、管理体系へ。
- 報酬: 消耗品・安物装備から、武具、魔法具、記録、固有遺物へ。
- 施設: 搬入口・倉庫から、居住区、兵舎、神殿、書庫、研究区、中枢へ。
- 物語: 方舟の外縁的痕跡から、閉鎖の理由、内部変質、最奥の真相へ。

## 3. 浅層

### 本来の用途

- 搬入口
- 外部連絡区
- 倉庫
- 家畜区
- 飼育区
- 廃棄区
- 下位作業区

### 現在の状態

- 荒廃している。
- 保存施設としての秩序はほぼ失われている。
- 生物は野生化・原始化している。
- 知性や組織性は弱く、巣・群れ・縄張りが中心。

### 敵の方向性

- 小型獣
- 昆虫型
- 粘性生物
- 低級ゴブリン系
- 腐敗した家畜・飼育生物
- 迷い込んだ弱い不死者

### 宝・報酬の方向性

- 薬、食料、灯火、鍵などの実用品
- 安物の武器・防具
- 小銭
- 破損した記録片
- 低階層用の通行具

### イベントの方向性

- 壊れた扉
- 錆びたレバー
- 使われなくなった倉庫
- 荒れた飼育区
- 初歩的な罠
- 方舟が本来は避難施設だったことを示す断片

## 4. 中層

### 本来の用途

- 居住区
- 工房
- 兵舎
- 礼拝所
- 物資管理区
- 下位保管区

### 現在の状態

- 住み着いた集団が縄張りを作っている。
- 元の種族や職能の名残があるが、閉鎖環境に適応して変質している。
- 部族化、武装化、儀礼化が進んでいる。

### 敵の方向性

- 武装した亜人
- オーク、トロール系
- 部族化したゴブリン上位種
- 亡者の兵士
- 変質した僧兵・守衛
- 群れを率いる中位個体

### 宝・報酬の方向性

- 実戦用の武具
- 鍵や通行証に相当するもの
- 古い貨幣
- 工房由来の道具
- 下位の魔法具
- 中層以降の施設に関わる記録

### イベントの方向性

- 兵舎跡
- 閉鎖された工房
- 祭壇または礼拝所
- 武器庫
- 縄張りの境界
- 部族化した住人の痕跡

## 5. 深層

### 本来の用途

- 書庫
- 神殿
- 研究施設
- 魔術師区
- 貴族区
- 高位保管区
- 管理区の外縁

### 現在の状態

- 知性、記録、信仰、魔術が歪んだ形で残っている。
- 単なる野生生物よりも、目的・儀礼・階級・魔術体系を持つ存在が多い。
- 方舟閉鎖の真相や中枢に近い情報が出始める。

### 敵の方向性

- 魔術師系の変質存在
- 司祭系の変質存在
- 高位不死者
- 精霊、悪霊、召喚体
- 知性的怪物
- 方舟の管理機構に由来する守護者

### 宝・報酬の方向性

- 魔法具
- 固有装備
- 重要文書
- 神殿・王家・魔術師団の遺物
- 鑑定や解呪に関わる品
- 最深層へ進むための権限・鍵・証

### イベントの方向性

- 書庫
- 封印室
- 神殿の奥所
- 研究室
- 管理記録
- 長大な時間経過を示す痕跡
- 方舟が閉じた理由に関わる断片

## 6. 最深層

### 本来の用途

- 方舟の中枢
- 封印制御区
- 管理座
- 高位保管庫
- 災厄前の世界記録

### 現在の状態

- 方舟の異常の根源に近い。
- 外界と内部時間のずれ、閉鎖の継続、魔法封印の変質に関わる。
- ここでは、単なる財宝よりも方舟の制御・開放・停止・真相が重要になる。

### 敵の方向性

- 方舟の守護者
- 管理者の成れの果て
- 高位魔術師・予言者・王族に由来する存在
- 封印そのものに結びついた存在
- 最終敵または最終判断を迫る存在

### 宝・報酬の方向性

- 方舟の中枢に関わる品
- 最終的な権限
- 災厄前の世界記録
- 最高位の魔法具・聖遺物
- ゲームの結末に関わる選択肢

### イベントの方向性

- 中枢制御
- 封印解除または再封印
- 方舟を開くか、閉じるか、制御するかの選択
- 方舟建造の真相
- 災厄と封印干渉の真相

## 7. 宝箱・保管物の配置ルール

- 宝箱は単なるランダムな財宝箱ではなく、階層用途に応じた保管物として扱う。
- 浅層では物資箱、工具箱、補給箱が中心。
- 中層では武器庫、工房保管箱、兵舎の収納が中心。
- 深層では封印箱、魔術師の保管箱、神殿の聖具庫、記録保管庫が中心。
- 最深層では中枢管理用の格納庫、制御具、最終遺物が中心。

## 8. 敵配置ルール

- 敵は強さだけで配置しない。
- 階層の本来用途、閉鎖後の変質、生態・社会・信仰との関係で配置する。
- 浅層は生存本能、巣、野生化を中心にする。
- 中層は部族化、武装化、縄張りを中心にする。
- 深層は知性、信仰、魔術、記録、階級を中心にする。
- 最深層は方舟の守護、封印、中枢、真相に関係する存在を中心にする。

## 9. 文言・固有名への影響

- アイテム名、魔法名、固有名は後で一覧化し、ユーザー確認後に正本化する。
- AIが仮に出す名称は採用名ではなく、差し替え前提の仮名とする。
- 階層名や重要施設名も、物語核と階層用途に合うものだけを採用候補とする。
- タイトルは仮置きでよい。階層設計上は、灰ではなく方舟・封印・中枢・地下・保管庫に接続する語を優先する。

## 10. 禁止事項

- 迷宮を毎回増殖・変化するランダム構造として扱わない。
- 深層に高価な宝がある理由を、単なるゲーム都合だけにしない。
- 浅層と深層の敵を、強さだけで入れ替えない。
- 「灰」を中核語として復活させない。ただしWizardry系の死の状態としての灰は、別問題として扱う。
- 日本語の固有名・美文をAI案だけで正本化しない。
