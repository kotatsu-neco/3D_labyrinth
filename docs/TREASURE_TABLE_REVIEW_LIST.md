# TREASURE_TABLE_REVIEW_LIST

この文書は、階層別の宝・報酬テーブルを後で調整するための確認リストである。
現時点の名称は内部用の仮IDであり、ゲーム内正式名ではない。

## 1. 運用方針

- アイテム名は後で一覧化し、まとめてユーザー確認・修正を行う。
- ここでは、名前ではなく「どの階層で何が出るべきか」を先に固定する。
- 宝箱は階層用途と施設用途に紐づける。
- 高価な宝を深層に置く理由は、方舟の管理区・神殿・書庫・研究区・中枢に近いからである。
