# REPORT v34

## 概要

v34では、街画面UIの器を追加した。ダンジョン内UIとの親和性を保つため、街画面も白枠、黒背景、モノスペース調、同じイベントウィンドウ構造で作成した。

## 実装内容

- 起動時に街画面を開く。
- 街から `迷宮へ` でダンジョンへ入る。
- 街に酒場、訓練場、宿屋、寺院、商店、冒険再開、遭難記録の導線を追加。
- 酒場はパーティー編成施設として表示。
- 訓練場は冒険者個人管理施設として表示。
- 訓練場の名簿からステータス詳細へ遷移可能。
- 宿屋・寺院・商店は施設画面の器を表示。
- キャンプに `周辺を探索する` を追加。
- キャンプに `冒険を中断する` を追加。
- 中断したパーティーは迷宮内の現在位置を保持する。
- 街の `冒険を再開する` から休止中パーティーを再開できる。
- 全滅時の自動街帰還は追加していない。
