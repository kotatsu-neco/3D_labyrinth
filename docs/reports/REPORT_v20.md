## 4. 反映した設計ルール

`docs/LAYER_DESIGN_RULES.md` に、フロア規模・マップ構造ルールを反映した。

- 標準フロアは20x20マスを基準とする。
- 通常階層で広げる場合も、原則24x24マスまでを上限目安とする。
- 30x30マスは標準化せず、特別階層・大型施設階層に限定する。
- 最深層や中枢階は、広さより密度を優先し、16x16〜20x20程度でもよい。
- 端と端がつながるトーラス型ループ構造は、現行仕様では採用しない。
- 方向感覚の揺さぶりは、ワープ、回転床、一方通行、隠し扉、似た構造の反復で表現する。
