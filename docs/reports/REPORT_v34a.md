# REPORT v34a

## 概要

v34aでは、v34全件検査で検出した街/中断/再開/全滅まわりの不整合を整理した。街画面UIの器は維持し、休止中パーティーと現在パーティーが同時に出発可能に見える問題、戦闘全滅時の不適切な `探索へ戻る` 文言、遭難記録未接続、未使用変数/関数を修正した。

## 反映内容

- 休止中パーティーがある場合、街の `迷宮へ` では無条件出発せず、`冒険を再開する` へ誘導する。
- v34aでは休止中パーティーは1件のみ保持する仕様として明記した。
- 戦闘全滅時の結果ボタンを `その場に残る` に変更した。
- 全員行動不能時、現在位置・メンバー・状態を `state.strandedParties` へ登録する最小処理を追加した。
- 遭難記録は街で表示できるが、自動帰還・自動救出・自動治療は行わない。
- `renderCampSearchWindow()` の重複注記を削除した。
- 未使用の `lastReturnReason` / `townReturnCount` / `townVisitCount` / `suspendedPartyLabel()` を削除した。
- `BUILD_VERSION` / `<title>` / HUD / キャッシュバスター / 戦闘CSS基準クラスを `v34a` に更新した。
