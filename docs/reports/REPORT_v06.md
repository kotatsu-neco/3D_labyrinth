# 灰の目録 3Dダンジョン試作 v06 作業報告

## 現在版

- 最新試作: v06

## v06での主な変更

- 宝箱と祭壇を `blocking: false` に変更し、通路中央を塞がない壁寄せ配置に変更。
- 宝箱と祭壇を小型化し、床面の通行スペースを確保。
- 扉のシェーダ内プロシージャル模様を無効化し、まだらに見えにくい単色寄りの表示に変更。
- レバー `lever01` に対応扉 `x5 y4` を設定し、操作時に開閉するように変更。
- 現在地にある宝箱・祭壇も `調べる` で反応するように変更。
- `index.html` のキャッシュバスターを `20260609-v06` に更新。
