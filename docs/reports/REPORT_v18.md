# 灰の目録 3Dダンジョン試作 最新作業報告

## 現在版

- 最新試作: v18
- 主対象: 探索イベントオブジェクトの専用ウィンドウ化

## 目的

v17でキャンプ・隊列・呪文確認・アイテム確認の管理系画面を拡張したため、v18では探索中のイベントオブジェクトを整理する。

宝箱だけが専用ウィンドウを持ち、石像・祭壇・床の紋様が一時表示だけで終わる状態を避ける。通常探索画面は基礎画面とし、イベントは専用ウィンドウで扱う方針に寄せる。

## 変更内容

- `index.html` のタイトルとキャッシュキーを v18 に更新。
- 祭壇 / 石像 / 床の紋様 / 罠検知時の床の印を専用イベントウィンドウで開くようにした。
- `inspectDungeonObject()` を整理し、宝箱以外の探索イベントも専用ウィンドウへ遷移するようにした。
- 床配置オブジェクトを現在マスから調べられるようにした。
- 罠床は通常時には反応させず、罠検知ONのときだけ専用ウィンドウ対象にした。
- 祭壇ウィンドウに `調べる` / `祈る` / `離れる` を追加。
- 石像ウィンドウに `調べる` / `触れる` / `離れる` を追加。
- 床の紋様ウィンドウに `調べる` / `踏み込む` / `離れる` を追加。
- それぞれの結果文は、説明文ではなく操作結果と確認事実が分かる短文にした。
- `docs/EVENT_OBJECT_RULES.md`、`docs/SCREEN_FLOW_RULES.md`、`docs/UI_STYLE_RULES.md` を v18 方針に合わせて更新。
