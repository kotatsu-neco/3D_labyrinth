# 灰の目録 3Dダンジョン試作 v13 作業報告

## 修正対象

ユーザー実機確認により、v12で以下の問題が報告された。

- スタート地点から動けない。
- ダンジョンが表示されない。

## 変更内容

### 1. 初期位置の変更

旧初期位置は `x1 y1 E` で、起動直後の視界情報が少なく、実機上で黒く見えやすかった。

v13では、起動直後から通路・配置物・進行方向が分かる確認用位置へ変更した。

- `START_POS = { x: 9, z: 10, dir: 3 }`
- `state.x` / `state.z` / `state.dir` の初期値を `START_POS` 参照へ変更。
- `初期位置` ボタンも `START_POS` へ戻るよう変更。

### 2. キャッシュキー更新

- `style.css?v=20260609-v13`
- `app.js?v=20260609-v13`
