# 灰の目録 3Dダンジョン試作 v08 作業報告

## 現在版

- 最新試作: v08
- 主対象: 固定グリッド式WebGLポリゴン3Dダンジョン表示
- 現在の重点: 宝箱のイベントウィンドウ化、イベントオブジェクトの設計ルール固定

## v08での主な変更

- 宝箱を、3D空間内で直接開ける対象ではなく、イベントウィンドウで操作する対象へ変更。
- 宝箱イベントウィンドウを追加。
  - 罠を調べる
  - 解除する
  - 開ける
  - 離れる
- 宝箱状態を保持する仮実装を追加。
  - 罠確認済み
  - 解除済み
  - 開封済み
- イベントウィンドウ表示中は、移動・旋回・調べる等のダンジョン操作を受け付けないように変更。
- `docs/EVENT_OBJECT_RULES.md` を追加し、視点上下操作は追加せず、詳細操作はイベントウィンドウに分離する方針を明文化。
- `index.html` のキャッシュバスターを `20260609-v08` に更新。

## 確認済み

- `node --check app.js` 通過。
- `index.html` が `style.css?v=20260609-v08` / `app.js?v=20260609-v08` を参照。
- `app.js` 内の初期メッセージが v08 になっている。
- 宝箱の「調べる」処理は、即時開封ではなく `openChestWindow()` を呼ぶ構成に変更。
- イベントウィンドウには、罠確認・解除・開封・離れるの操作ボタンを実装。
- 開封後は宝箱の3D表示も開封済み表示へ変わる構成を維持。
- `docs/EVENT_OBJECT_RULES.md` を同梱。
