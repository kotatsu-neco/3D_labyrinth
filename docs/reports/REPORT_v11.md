# 灰の目録 3Dダンジョン試作 v11 作業報告

## 目的

v11では、宝箱を固定視点でも扱いやすくし、将来の宝箱画像差し替えに備える。あわせて、未確認だったイベントオブジェクト類を簡易配置し、今後の検査対象を増やした。

## 変更内容

### 1. 宝箱の同一マス調査

- 宝箱は、同じマスに立っていれば向きに関係なく `調べる` でイベントウィンドウを開けるようにした。
- 下を見る操作は追加しない方針を維持。

### 2. 宝箱画像の差し替え準備

- イベントウィンドウ内の宝箱表示を、CSS描画から外部SVG画像参照へ変更。
- 仮アセットを追加。
  - `assets/ui/chest_closed.svg`
  - `assets/ui/chest_open.svg`

### 3. 確認用オブジェクト追加

- 石像
- 魔法陣
- 罠床風の床
- 敵影

これらは本格実装ではなく、3Dダンジョン内での見え方・配置方針を確認するための仮表示である。

### 4. 文書更新

- `docs/EVENT_OBJECT_RULES.md` に以下を追記。
  - 宝箱画像の差し替え方針
  - 同一マスの宝箱調査ルール
- `docs/VISUAL_AND_LIGHTING_RULES.md` を同梱。
- `README.md` / `REPORT.md` をv11に更新。
