# 作業報告書 v01

## 確認済み

- `index.html`, `style.css`, `app.js`, `README.md` を作成した。
- `node --check app.js` でJavaScriptの構文チェックを通した。
- `app.js` は外部ライブラリなしでWebGLを初期化する構成にした。
- ダンジョン描画はCanvas 2Dの擬似3Dではなく、WebGLの三角形ポリゴンで壁・床・天井を描画する方式にした。
- 固定グリッド式の `x`, `z`, `dir` による位置管理を実装した。
- 前進、後退、左旋回、右旋回、調べる、初期位置、簡易マップボタンを実装した。
- 閉じた扉を `調べる` で開ける処理を実装した。
- 階段、イベント床のマーカー表示を実装した。

## 未確認

- iPhone実機SafariでのWebGL描画は、この環境では直接確認していない。
- headless Chromiumでのスクリーンショット取得は試したが、この実行環境ではGPU/DBus関連エラーで完了しなかったため、描画スクリーンショット確認は未確認。
- 低スペック端末でのフレームレートは未確認。
- すべてのモバイルブラウザでのタッチ挙動は未確認。

## 想定

- iOS SafariでWebGLが有効なら表示できる想定。
- `file://` 直開きでも動く可能性は高いが、端末・ブラウザによってはローカルサーバー配信の方が安定する想定。
