# 灰の目録 3Dダンジョン試作 最新作業報告

## 現在版

- 最新試作: v17
- 主対象: キャンプ内サブ画面、隊列、呪文確認、アイテム確認の基盤

## 目的

v16で追加したキャンプ画面とキャラクター詳細を、次の管理系画面へつなげる。

v17では、実際の装備変更・呪文使用・アイテム使用までは行わず、キャンプから各専用ウィンドウへ移動し、必要な情報を確認できる状態にする。

## 変更内容

- `index.html` のタイトルとキャッシュキーを v17 に更新。
- キャンプ内コマンドを整理。
  - メンバーを見る
  - 隊列
  - 装備確認
  - 呪文
  - アイテム
  - 出る
- 隊列画面を追加。
  - 6人の並びを表示。
  - 上下ボタンで並び順を変更。
  - 先頭3人を前衛、後ろ3人を後衛として反映。
- 通常探索画面の `隊列` ボタンを、隊列専用ウィンドウへのショートカットに変更。
- 呪文画面を追加。
  - キャラクター選択後、`MAGE / PRIEST` の7レベル別使用回数を表示。
  - 呪文使用はまだ行わない。
- アイテム画面を追加。
  - キャラクター選択後、所持品を表示。
  - 使用・受け渡し・破棄はまだ行わない。
- 装備確認画面を追加。
  - 各キャラクターの所持品概要を一覧表示。
  - 装備変更はまだ行わない。
- キャラクター詳細画面から `呪文` / `アイテム` へ移動できるようにした。
- `docs/SCREEN_FLOW_RULES.md` と `docs/UI_STYLE_RULES.md` を v17 方針に合わせて更新。
