# REPORT_v27

## 対象

- プロジェクト: 暗澹の石櫃 / Stone Casket of Gloom
- バージョン: v27
- ベース: v26

## 作業概要

v27では、v26までに接続したランダム遭遇、戦利品、宝箱報酬、基礎呪文/道具効果、簡易レベルアップを維持し、戦闘中の呪文・回復道具に対象選択を追加した。

これにより、v26時点で自動対象だった「火花」「治癒」「HEALING HERB」を、戦闘画面の入力状態として明示的に扱うようにした。

## 変更内容

### バージョン更新

- `app.js` の `BUILD_VERSION` を `v27` に変更。
- `index.html` の `<title>` を `暗澹の石櫃 v27` に変更。
- 画面上部HUDを `暗澹の石櫃 v27` に変更。
- キャッシュバスターを `20260612-v27` に変更。
- 戦闘ウィンドウのCSS基準クラスを `battle-window-v27` に更新。

### 呪文の対象選択

- `MAGE 火花` は、呪文選択後に敵対象選択へ遷移するよう変更。
- `PRIEST 治癒` は、呪文選択後に味方対象選択へ遷移するよう変更。
- 味方対象選択では、生存中かつ負傷しているメンバーのみ選択可能。
- 対象選択から呪文選択へ戻れるようにした。

### 道具の対象選択

- `HEALING HERB` は、道具選択後に味方対象選択へ遷移するよう変更。
- 使用対象は、生存中かつ負傷しているメンバーのみ。
- 回復対象がいない場合、使用・消費されない。
- 戦闘中に未接続のアイテムは選択不能にした。

### 状態遷移の整理

追加・整理した入力状態:

- `spell_target_enemy`
- `spell_target_ally`
- `item_target_ally`

既存の `spell_select` / `item_select` / `target_select` と分け、戻る操作で一段前に戻るようにした。
