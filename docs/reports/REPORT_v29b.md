# REPORT_v29b

## 概要

- 対象: 暗澹の石櫃
- バージョン: v29b
- ベース: v29a
- 主目的: キャンプ系ウィンドウのフォントサイズを、ステータス詳細画面のサイズ感へ寄せる。

## 変更内容

v29bでは、ユーザー提供スクリーンショットで確認されたキャンプトップと同系統の画面について、文字サイズがステータス詳細画面より大きく見える問題に対応した。

### 反映箇所

- `style.css`
  - `.camp-window-panel` 系のヘッダー、プロンプト、コマンドボタン、三角マーカー、CAMP表示、注記欄を調整。
  - `.character-detail-panel:not(.status-detail-v23g-panel)` 系のアイテム/呪文/対象選択画面のコマンドボタンを調整。
  - 狭幅画面向け `@media (max-width: 390px)` でも、ステータス詳細画面に近いサイズへ調整。
- `app.js`
  - `BUILD_VERSION` を `v29b` に更新。
  - 戦闘ウィンドウ基準クラスを `battle-window-v29b` に更新。
- `index.html`
  - `<title>`、HUD、キャッシュバスターを `v29b` に更新。
- `README.md` / `REPORT.md` / `docs/reports/REPORT_v29b.md`
  - v29b内容へ更新。

### フォントサイズ方針

- ステータス詳細画面の既存値を基準にする。
- コマンドボタンは通常幅 `16px`、狭幅 `15px` に寄せる。
- キャンプトップの大きすぎるプロンプト `22px` 系は、通常幅 `17px`、狭幅 `16px` に縮小する。
- ステータス詳細画面自体は変更対象外とし、セレクタで `status-detail-v23g-panel` を除外した。
