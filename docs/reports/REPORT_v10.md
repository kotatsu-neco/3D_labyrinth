# 灰の目録 3Dダンジョン試作 v10 作業報告

## 目的

宝箱イベントウィンドウ内の、選択肢とメッセージの位置関係を再設計する。FC/NES版Wizardryの宝箱画面を参考に、黒地・白枠・枠分割・コマンドリスト・パーティ表示という雰囲気を、スマホ縦画面へ落とし込む。

## 変更内容

### 1. イベントウィンドウ構造

`app.js` の `renderChestWindow()` を変更した。

- 上段: メッセージ枠
- 中段左: オブジェクト枠
- 中段右: コマンド枠
- 下段: パーティ枠

### 2. コマンド表示

`style.css` を変更した。

- 2列グリッドボタンを廃止。
- 縦リスト式コマンドに変更。
- 各行の左に赤いカーソル記号を表示。
- タップしやすい最小高さは維持。

### 3. 文言抑制の維持

以下は表示しないまま維持した。

- 宝箱説明文
- 罠ヒント
- 未確認/未解除などの状態列挙
- 成功率や危険予告

### 4. 設計文書

`docs/UI_STYLE_RULES.md` を追加した。
