# タイトル未定 地下方舟3Dダンジョン試作 v21 作業報告

## 1. 作業目的

v21では、敵画像を戦闘画面向けに低解像度・減色処理し、確認用素材としてパッケージに追加した。
戦闘本実装や敵遭遇ウィンドウはまだ行わず、素材処理と設計文書の同梱を主目的とした。

## 2. 追加した文書

### `docs/ENEMY_IMAGE_PROCESSING_RULES.md`

- 敵画像を高解像度のまま戦闘画面へ使わない方針を記録。
- v21処理ルールとして、160x160 PNG、80px相当まで縮小後に最近傍で拡大、色数目標24色、可能な範囲で背景透過を記録。
- 処理済み画像も最終採用ではなく確認用候補とした。

### `docs/PARAMETER_DESIGN_RULES.md`

- アイテム、呪文、敵、遭遇、宝、状態異常のパラメータ設計方針を同梱。
- アイテムはスタックさせず、すべて1個ずつの所持品インスタンスとして扱う方針を記録。
- アイテムに物理重量 `weight` は持たせない方針を記録。

## 3. 追加した画像素材

### `assets/enemies/lowres/`

- 通常敵画像28点を160x160 PNGへ処理。
- 元画像はおおむね1254x1254 px。
- 実質ディテールは80px相当まで落としている。
- 画像はアプリ本体にはまだ組み込んでいない。

### `assets/enemies/silhouettes_lowres/`

- シルエット画像28点を160x160 PNGへ処理。
- 通常敵画像と対応する確認用素材。

### `docs/assets/enemy_assets_lowres_contact_sheet_v21.png`

- 処理済み敵画像の確認用コンタクトシート。
- 通常画像とシルエットを並べて確認する用途。

## 4. 追加したデータ

### `data/enemy_assets_v21.json`

- 処理済み敵画像56件のメタデータ。
- 通常敵28件、シルエット28件。
- 元ファイル名、出力ファイル名、元サイズ、出力サイズ、処理ルール、ファイルサイズを記録。

## 5. アプリ本体への変更

- `BUILD_VERSION` を `v21` に更新。
- HTMLタイトル・HUD表示・キャッシュバスターをv21へ更新。
- 3D描画、移動、調査、罠検知、キャンプ、隊列、宝箱イベントのロジックは変更していない。
- 敵画像はまだ画面表示に使っていない。
