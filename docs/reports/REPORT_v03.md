# 灰の目録 3Dダンジョン試作 v03 作業報告

## 目的
v02bの実機確認コメントを受け、UI全体には踏み込まず、ダンジョン表示・移動感・意図しない選択防止だけを修正した。

## 反映内容

### 1. 扉表示の明確化
- 閉じた扉を、黒く潰れやすいブロック表示から、薄い扉板 + 石枠の表示に変更。
- 扉面にはWebGLシェーダ側で木板・縦目地・金属帯・縁取りの簡易プロシージャル模様を付与。
- 扉セルにも床・天井を描画し、黒い空白に見えにくい構成へ変更。

### 2. 通路幅の追加拡張
- 描画用セル幅 `CELL` を `1.5` から `1.75` に変更。
- 内部グリッド座標、マップ配列、当たり判定は維持。

### 3. 歩行・旋回・衝突時の揺れ縮小
- 歩行時の上下動を `0.045/-0.018` から `0.026/-0.010` に縮小。
- 旋回時の横揺れを `0.018` から `0.010` に縮小。
- 壁衝突時のバンプ移動量を `0.085` から `0.052` に縮小。
- 壁衝突時の上下動を `0.025` から `0.012` に縮小。

### 4. 連打時の意図しない選択抑止
- ボタン入力を `click` 直接処理から `pointerdown` 主体に変更。
- `preventDefault()` / `stopPropagation()` を追加。
- `el.blur()` を追加し、操作後にボタンフォーカスが残りにくいようにした。
- CSSに `-webkit-user-select: none` / `user-select: none` / `-webkit-touch-callout: none` を追加。
- 短時間の二重発火抑止として90msの入力ガードを追加。

### 5. キャッシュ対策
- `index.html` の読み込みを `style.css?v=20260609-v03` / `app.js?v=20260609-v03` に更新。
- 画面タイトル・HUD・初期メッセージをv03に更新。
