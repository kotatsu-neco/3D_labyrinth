# タイトル未定 地下方舟3Dダンジョン試作 v19 作業報告

## 作業目的

v19では、直近で決まった物語核と階層設計方針を、試作物と設計文書へ反映した。
旧仮題「灰の目録」は、Wizardry系の死亡段階における「灰」と混同するため、現行タイトル表示から外した。

## 実施内容

### 1. タイトル表示の仮置き化

- `index.html` のタイトルを `タイトル未定 地下方舟3Dダンジョン試作 v19` に変更。
- HUD表示を `タイトル未定 / 地下方舟3Dダンジョン試作 v19` に変更。
- READMEの表題も同じ方針に更新。

### 2. 階層メタ情報の追加

`app.js` に、現在階層の設計上の位置づけを示すメタ情報を追加した。

- B1F: 浅層
- zone: 外縁部
- role: 搬入口・倉庫・飼育区に近い確認階層

現時点では、階層移動や敵出現には未接続。
HUDの位置表示には `B1F 浅層` として反映している。

### 3. 物語核・階層設計文書の同梱

- `docs/STORY_CORE.md`
- `docs/LAYER_DESIGN_RULES.md`

をzip内へ同梱した。
GitHubに先行記録した方針と整合する内容である。

### 4. アイテム名確認リストの追加

`docs/ITEM_NAME_REVIEW_LIST.md` を追加した。
現在アプリ内に仮置きされている英語アイテム名を一覧化し、後でまとめて修正できるようにした。

### 5. 旧仮題表記の整理

現行の設計文書について、旧仮題「灰の目録」の表記を原則として `本作` または `タイトル未定 3Dダンジョン試作` に置換した。
過去バージョンの報告書は履歴なので、旧仮題表記を残している。
