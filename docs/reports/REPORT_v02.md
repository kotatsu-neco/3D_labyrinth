# 灰の目録 3Dダンジョン試作 v02 作業報告

## 目的

ユーザー実機スクリーンショット確認後の指摘に基づき、UI本体には大きく触れず、3Dダンジョン部分だけを改善した。

対象指摘:

1. 一歩一歩の動きにメリハリをつける
2. 通路幅を1.5倍くらい広げる
3. 通路、壁、天井を視認しやすくする
4. 壁は壁っぽいテクスチャが貼れるか試す

## 変更内容

### 1. 歩行アニメーション

`app.js`

- `startMoveAnimation()` の移動時間を 240ms に変更。
- `easeInOutCubic()` を採用。
- `visual.stepBob` を追加し、歩行中に軽い上下動を付与。
- 終端側に軽い沈み込みを入れ、着地感を出した。

### 2. 壁衝突時の動き

`app.js`

- `startBumpAnimation()` を追加。
- 壁または閉じた扉に進もうとしたとき、短く前後に揺れる動きを追加。

### 3. 旋回アニメーション

`app.js`

- 旋回時間を 190ms に変更。
- `visual.turnLean` を追加し、旋回中に軽い横揺れを付与。

### 4. 通路幅拡張

`app.js`

- `CELL = 1.5` を追加。
- マップの論理座標は維持し、描画座標のみ1.5倍に変換。
- カメラ座標、壁、床、天井、扉、階段、イベント床マーカーも `CELL` 基準に変更。

### 5. 壁・床・天井の視認性改善

`app.js`

- 頂点属性に `aUV` と `aSurface` を追加。
- WebGL fragment shaderで壁・床・天井ごとにテクスチャを切り替える構成に変更。
- `makeWallTexture()` で石積み風の壁テクスチャをCanvas生成。
- `makeFloorTexture()` で床石風テクスチャをCanvas生成。
- `makeCeilingTexture()` で天井用テクスチャをCanvas生成。
- 霧の開始距離を奥へずらし、近距離の壁面が黒く潰れすぎないよう調整。
