# ENEMY_AND_TREASURE_RULES

この文書は、地下方舟型迷宮における敵・宝・施設配置の設計ルールである。
`docs/STORY_CORE.md` と `docs/LAYER_DESIGN_RULES.md` を前提とする。

## 1. 基本方針

- 敵と宝は、単なる強さ・価格順ではなく、階層の本来用途と閉鎖後の変質に基づいて配置する。
- 浅層ほど生存本能・野生化・原始化を重視する。
- 深層ほど知性・信仰・魔術・記録・管理機構を重視する。
- 宝箱はランダムな財宝箱ではなく、方舟内の保管物として扱う。
- 敵名、アイテム名、魔法名、固有施設名は仮名扱いとし、後で一覧化してユーザー確認後に正本化する。

## 2. 敵データの分類軸

敵を登録する際は、最低限以下の軸を持たせる。

| 軸 | 内容 |
|---|---|
| layer_band | shallow / middle / deep / core |
| ecology | 獣、虫、粘性生物、亜人、不死、精霊、守護者など |
| intelligence | low / medium / high / guardian |
| society | 単体、群れ、巣、部族、信仰、管理機構 |
| role | 序盤雑魚、群れ、番兵、術者、重戦士、守護者など |
| ac_profile | 低AC/高ACではなく、当たりにくさの傾向として管理する |
| treasure_profile | 落とす宝・守る保管物の傾向 |
| name_status | 仮名 / 要改名 / 正本化済み |

## 3. 浅層の敵・宝

### 敵

浅層は、方舟の外縁、搬入口、倉庫、飼育区、廃棄区に近い。
敵は原始化・野生化したものを中心にする。

候補:

- 小型獣
- 昆虫型
- 粘性生物
- 低級ゴブリン系
- 腐敗した飼育生物
- 弱い不死者

配置方針:

- 単体または小集団。
- 知性よりも生存本能を優先。
- 武装や魔法は少ない。
- 宝は少額・実用品中心。

### 宝

- 薬、食料、灯火などの消耗品
- 安物装備
- 小銭
- 破損した記録片
- 低階層用の鍵または通行具

## 4. 中層の敵・宝

### 敵

中層は、居住区、兵舎、工房、礼拝所、物資管理区に近い。
敵は部族化・武装化・縄張り化したものを中心にする。

候補:

- 武装した亜人
- オーク、トロール系
- 部族化したゴブリン上位種
- 亡者の兵士
- 変質した僧兵・守衛
- 群れを率いる中位個体

配置方針:

- 群れや隊列を持つ。
- 武器・盾・防具を持つ敵を増やす。
- 簡単な戦術や役割分担を持たせる。
- 宝は武具、工具、鍵、古い貨幣を中心にする。

### 宝

- 実戦用の武具
- 工房由来の道具
- 兵舎の保管物
- 中層以降の鍵・通行証
- 下位魔法具
- 施設に関わる記録

## 5. 深層の敵・宝

### 敵

深層は、書庫、神殿、研究施設、魔術師区、貴族区、高位保管区に近い。
敵は知性・信仰・魔術・記録が歪んだ形で残ったものを中心にする。

候補:

- 変質した魔術師
- 変質した司祭
- 高位不死者
- 精霊、悪霊、召喚体
- 知性的怪物
- 方舟の管理機構に由来する守護者

配置方針:

- 単なる肉弾敵だけにしない。
- 魔法、状態異常、耐性、特殊行動を持つ敵を増やす。
- 宝は魔法具、固有装備、重要文書、聖遺物を中心にする。

### 宝

- 魔法具
- 固有装備
- 重要文書
- 神殿・王家・魔術師団の遺物
- 鑑定や解呪に関わる品
- 最深層に進むための証・鍵・権限

## 6. 最深層の敵・宝

### 敵

最深層は、方舟の中枢、封印制御区、管理座に近い。
敵は方舟そのもの、封印、管理権限、閉鎖の真相に関係する存在を中心にする。

候補:

- 方舟の守護者
- 管理者の成れの果て
- 高位魔術師・予言者・王族に由来する存在
- 封印そのものに結びついた存在
- 最終敵または最終判断を迫る存在

配置方針:

- 敵の強さだけでなく、最奥の意味に関係させる。
- 戦闘だけでなく、開放・再封印・制御などの選択につながる余地を残す。

### 宝

- 方舟の中枢に関わる品
- 最終的な権限
- 災厄前の世界記録
- 最高位の魔法具・聖遺物
- 結末分岐に関わるもの

## 7. 宝箱種別

宝箱・保管物は以下の種別で分類する。

| 種別 | 主な階層 | 内容 |
|---|---|---|
| supply_cache | 浅層 | 消耗品、食料、簡易道具 |
| tool_box | 浅層〜中層 | 工具、鍵、解除道具 |
| armory_chest | 中層 | 武器、防具、兵装 |
| workshop_case | 中層 | 工房道具、修理品、素材 |
| reliquary_case | 深層 | 神殿・王家・魔術師団の遺物 |
| sealed_coffer | 深層 | 魔法具、封印物、鑑定対象 |
| archive_box | 深層〜最深層 | 記録、文書、地図、証 |
| core_vault | 最深層 | 中枢権限、最終遺物 |

## 8. v20時点の実装方針

- v20では敵画像の棚卸しと分類を行う。
- v20では戦闘本実装は行わない。
- v20では敵画像をアプリに組み込まない。
- v20では敵名・アイテム名を正本化しない。
- 次段階で、敵遭遇ウィンドウの枠だけを作る。
