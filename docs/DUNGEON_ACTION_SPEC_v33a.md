# DUNGEON_ACTION_SPEC_v33a

## 目的

v33aは、v33で入れた暫定仕様を整理し、街機能実装前のダンジョン内行動を最終形に近づける修正版である。

ユーザー指示により、以下を撤回する。

- キャンプ内の「休ませる」
- キャンプ内の暫定「帰還」
- ASLEEP / AFRAID をキャンプで回復させる仕様

## 採用方針

### キャンプ

キャンプは迷宮内で隊を整える画面とする。

採用コマンドは以下。

- メンバーを見る
- 隊列
- 装備
- 呪文
- アイテム
- 出る

以下は採用しない。

- 休ませる
- 帰還

### ASLEEP / AFRAID

`ASLEEP / AFRAID` は戦闘中の一時的な行動不能として扱う。
戦闘終了時に `OK` へ戻る。

キャンプ内では回復対象にしない。
PRIEST呪文にも専用回復を置かない。

宝箱罠では `ASLEEP / AFRAID` を発生させない。罠は迷宮内で継続する状態異常・ダメージに限定する。

### 迷宮内PRIEST呪文

v33aで扱うキャンプ内PRIEST呪文は以下。

| 状態 | 処置 |
| --- | --- |
| HP負傷 | 治癒 |
| POISONED | 毒消し |
| PARALYZED | 麻痺回復 |
| STONED | 石化解除 |
| OUT | 気付け |
| DEAD | 蘇生 |
| ASHES | 灰から蘇生 |
| LOST | 迷宮内回復不可 |

### 全員行動不能

全員行動不能判定は維持する。
ただし、街UI未実装段階では、暫定的に入口へ戻す処理を入れない。

街UI実装後、以下へ接続する。

- 救出
- 寺院
- 蘇生費用
- 灰化蘇生
- ロスト回収可否

### 街接続

v34では街画面のUI設計・実装を行う。
その後、街機能を仕様書として書き下ろし、本実装はCodexで行う。

## v33aで実装したこと

- `BUILD_VERSION = "v33a"` へ更新。
- 戦闘ウィンドウ基準クラスを `battle-window-v33a` へ更新。
- キャンプメニューから「休ませる」を削除。
- キャンプメニューから「帰還」を削除。
- キャンプ内PRIEST呪文から「覚醒」を削除。
- 戦闘終了時に `ASLEEP / AFRAID` を回復する処理を追加。
- 全員行動不能画面から暫定救出ボタンを削除。

## v33aで実装しないこと

- 街画面。
- 街への帰還遷移。
- 寺院。
- 宿屋。
- 店。
- 救出・回収処理。
- 後衛攻撃。
- 未鑑定・鑑定・呪い・解呪。
