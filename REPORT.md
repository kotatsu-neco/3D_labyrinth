# 灰の目録 3Dダンジョン試作 最新作業報告

## 現在版

- 最新試作: v12
- 主対象: 固定グリッド式WebGLポリゴン3Dダンジョン表示
- 現在の重点: Wizardry風UIへの寄せ、罠床表示方針、パーティ状態表示

## v12での主な変更

- 敵影を削除。
- 罠床は通常非表示にし、検知状態でのみ淡い青白い表示に変更。
- テスト用 `罠検知` ボタンを追加。
- 宝箱イベントウィンドウ内のパーティ表示を大きめに調整し、MPと状態表示を追加。
- 通常探索画面のパーティ表示を、宝箱イベントウィンドウ寄りの黒地・白枠・表形式へ変更。
- `docs/UI_STYLE_RULES.md` と `docs/EVENT_OBJECT_RULES.md` を更新。

## 確認済み

- `node --check app.js` 通過。
- `index.html` は `style.css?v=20260609-v12` / `app.js?v=20260609-v12` を参照している。
- 敵影オブジェクトは `DUNGEON_OBJECTS` から削除済み。
- 罠床は `trapDetectionActive` が有効な場合だけ描画する。
- 通常探索画面と宝箱イベントウィンドウのパーティ表示は、NAME / CLASS / HITS / MAGIC / STATUS の列構成になっている。
