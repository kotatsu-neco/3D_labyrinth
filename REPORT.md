# REPORT v34b

## 目的

v34aの街UI器を、確定した `town_spec_v01e` の街機能仕様に合わせて修正した。特に、宿屋・寺院・商店の性質をUI上で誤解しないようにした。

## 修正内容

### 宿屋

- `宿屋` を単なる施設スタブから、個人選択画面へ変更した。
- 対象キャラクター選択後、部屋ランク選択画面へ進む導線を追加した。
- 部屋ランクは以下を表示する。
  - 馬小屋
  - 大部屋
  - 二人部屋
  - 個室
  - 王室
- 宿屋注記に、個人宿泊・本人所持金支払い・宿泊後レベルアップ判定を明記した。

### 寺院

- `寺院` を単なる施設スタブから、処置対象選択画面へ変更した。
- 処置対象選択後、支払者選択画面へ進む導線を追加した。
- `お金を集める` の器を追加した。
- 寺院注記に、未回収者は対象外、処置対象本人から支払わない、加齢は1d52週であることを明記した。

### 商店

- `商店` を単なる施設スタブから、操作選択画面へ変更した。
- 操作は以下。
  - 買う
  - 売る
  - 鑑定
  - 解呪
  - お金を集める
  - 街へ戻る
- `買う` / `売る` / `鑑定` / `解呪` は、最初に対象キャラクター選択画面へ進む。
- 商店注記に、個人単位、暗黙の共有財布禁止、対象者本人の所持金・所持品だけを扱うことを明記した。
- 商店の `お金を集める` は、買い物担当者へ金を集中させる補助機能として配置した。

### 酒場・訓練場

- 酒場注記を、金分配=均等配分として明確化した。
- 訓練場注記を、個人管理施設であり、後続実装では入場時に街編成パーティー解散確認が必要であることへ更新した。
