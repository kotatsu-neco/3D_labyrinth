# 灰の目録 3Dダンジョン試作 最新作業報告

## 現在版

- 最新試作: v07
- 主対象: 固定グリッド式WebGLポリゴン3Dダンジョン表示
- 現在の重点: 扉色、開扉方向、下り階段の配置・作図

## v07での主な変更

- 扉色をv06より暗い茶色へ調整。
- 開扉後の左右扉板を同じ側へ開く配置に統一。
- 下り階段を角配置から `x8 y8` へ移動。
- 下り階段マスでは通常床を描かず、床開口部・下降段・暗がり・側壁を描画する方式へ変更。
- `docs/DUNGEON_DESIGN_RULES.md` を同梱し、階段は原則として角に置かない方針を明文化。
- `index.html` のキャッシュバスターを `20260609-v07` に更新。

## 確認済み

- `node --check app.js` 通過。
- `index.html` が `style.css?v=20260609-v07` / `app.js?v=20260609-v07` を参照。
- `app.js` 内の初期メッセージが v07 になっている。
- map上の下り階段 `S` は角の `x10 y1` から `x8 y8` へ移動。
- 下り階段タイルでは標準床描画をスキップする処理を追加。
