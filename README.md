# タイトル未定 地下方舟3Dダンジョン試作 v23b

タイトルは未確定です。v23bでは、戦闘LOG枠へトースト風のタイプライター表示を追加しました。

## 現在の実行方法

`index.html` をブラウザで開く。

GitHub Pages等で配布する場合は、キャッシュ更新のため `style.css` と `app.js` に `20260610-v23b` のクエリを付けています。

## v23b 変更点

- 戦闘LOG枠にトースト風の文字送り表示を追加。
- 最新ログを1文字ずつ表示するタイプライター処理を追加。
- 文字送り速度を `BATTLE_LOG_TYPE_SPEED_MS` で調整可能にした。
  - v23b初期値: `18ms`
  - `0` にすると即時表示。
- LOG枠タップ、またはコマンド入力時に、表示中ログを即時全文表示するようにした。
- 表示中に次ログが来た場合はキューに積み、現在ログの表示完了後に順番に表示する。
- 古いログは最大3行分だけ残し、最新ログを強調する。
- `BUILD_VERSION` とキャッシュバスターを `v23b` に更新。
- `REPORT.md` / `docs/reports/REPORT_v23b.md` を追加。
