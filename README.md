# 灰の目録 3Dダンジョン試作 v15c

WebGLで描画する、固定グリッド式の軽量ポリゴン3Dダンジョン試作です。

## 起動方法

`index.html` をブラウザで開いてください。
GitHub Pages等に配置する場合は、フォルダ内のファイル一式をそのまま配置します。

## 操作

- ↑: 前進
- ↓: 後退
- ←: 左旋回
- →: 右旋回
- 調べる: 前方または現在マスの配置物を調べる / 閉じた扉を開く / 壁レバーを操作する
- パーティカード: 簡易ステータスウィンドウを開く
- 初期位置: 開始位置に戻る
- 簡易マップ: デバッグ用マップ表示
- 罠検知: テスト用。検知状態のときだけ罠床を淡く表示する

## v15c 修正点

- v15の全件検査結果とv15b実機確認後の文言指摘を受けた整理版。
- 3D表示領域の最小高さを `280px` 固定から `clamp(300px, 46svh, 360px)` に変更。
- 起動直後の説明トーストを削除。
- 通常探索時の一時表示文言を、短すぎる単語表示ではなく、操作結果と確認事実が分かる短文へ修正。
- レバー操作時は、上げた/下げたの動作と、どこかで音がしたことを表示。
- 移動・旋回時の逐次トーストを削除し、通常探索画面の沈黙を増やした。
- 簡易マップのCSSを動的挿入から `style.css` 定義へ移動。
- キャッシュバスターを `20260609-v15c` に更新。

## v15から継続している内容

- 通常探索画面から常設メッセージ枠を削除。
- パーティ表示を3列2行のカード型へ変更。
- 通常カードの表示項目を `名前 / 職業 / HP / AC / 状態` に整理。
- `MP` / `MAGIC` 表示を通常探索画面と宝箱イベント内パーティ表示から削除。
- キャラクターカードをタップすると、簡易ステータス専用ウィンドウを開く。
- 宝箱の罠解除成功担当を盗賊の `ミラ` に修正。
