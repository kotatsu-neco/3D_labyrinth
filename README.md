# 暗澹の石櫃 town_impl_v01

Wizardry 風スマホWeb RPGプロトタイプです。`town_impl_v01` では、v34bで用意した街UIの器に、`town_func_spec_v01e` に基づく街機能の最小実処理を接続しました。

## 起動

- `index.html` をブラウザで開く。
- 起動時は街画面から開始する。
- `迷宮へ` は、現在街パーティーに1人以上いる場合だけ出発する。

## 実装内容

- `state.adventurers` / `state.parties` / `state.strandedRecords` を導入。
- `PARTY_MEMBERS` は現在操作中パーティーの互換表示用配列として同期。
- 戦闘勝利時はEXP/GOLD配分のみ行い、レベルアップしない。
- 宿屋宿泊後にだけ、1回の宿泊につき最大1レベルのレベルアップ判定を行う。
- 酒場で、街にいる冒険者の加入、現在街パーティーから外す、均等な金分配、解散を実行。
- 訓練場で、入場前の街パーティー解散確認、新規作成、転職、2段階削除を実行。
- 宿屋で、個人単位の部屋ランク宿泊、本人支払い、HP/呪文回復、加齢を実行。
- 寺院で、処置対象と支払者を分離し、支払者の所持金から処置費用を支払う。
- 寺院/商店の `お金を集める` は、現在街パーティーの金を選択した1人へ集中する。
- 商店で、対象者本人の所持金・所持品だけを対象に、買う/売る/鑑定/解呪を実行。
- `冒険を中断する` は帰還扱いせず、位置・状態・所持品・金・HP・呪文回数を保持して街操作へ戻す。
- 全員行動不能時は `StrandedRecord` を作り、メンバーを `STRANDED` として迷宮内に残す。
- キャンプの `周辺を探索する` で、同一階層かつマンハッタン距離1以内の遭難者を確定発見し、空き枠分だけ `CARRIED` として搬送する。
