# 灰の目録 3Dダンジョン試作 v14

WebGLで描画する、固定グリッド式の軽量ポリゴン3Dダンジョン試作です。

## 起動方法

`index.html` をブラウザで開いてください。
GitHub Pages等に配置する場合は、フォルダ内のファイル一式をそのまま配置します。

## 操作

- ↑: 前進
- ↓: 後退
- ←: 左旋回
- →: 右旋回
- 調べる: 前方または壁際の配置物を調べる / 閉じた扉を開く / 壁レバーを操作する
- 初期位置: 開始位置に戻る
- 簡易マップ: デバッグ用マップ表示
- 罠検知: テスト用。検知状態のときだけ罠床を淡く表示する

## v14 修正点

- v12/v13で発生した「ダンジョンが表示されない」「一歩も動けない」「向きも変えられない」問題を修正。
- 原因は `detectTrapBtn` に割り当てていた `toggleTrapDetection` が未定義で、起動時に `ReferenceError` が発生していたこと。
- `toggleTrapDetection()` を実装し、罠検知トグルでシーンを再構築するようにした。
- `bindButton()` に、ボタンまたはハンドラが欠けていてもアプリ全体を停止させない防御を追加。
- キャッシュバスターを `20260609-v14` に更新。
