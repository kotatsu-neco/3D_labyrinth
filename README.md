# 灰の目録 3Dダンジョン試作 v13

WebGLで描画する、固定グリッド式の軽量ポリゴン3Dダンジョン試作です。

## 起動方法

`index.html` をブラウザで開いてください。
GitHub Pages等に配置する場合は、フォルダ内のファイル一式をそのまま配置します。

## 操作

- ↑: 前進
- ↓: 後退
- ←: 左旋回
- →: 右旋回
- 調べる: 前方または同一マスの宝箱・壁際配置物を調べる / 閉じた扉を開く / 壁レバーを操作する
- 初期位置: 確認用の開始位置に戻る
- 簡易マップ: デバッグ用マップ表示
- 罠検知: テスト用。検知時のみ罠床を青白く表示

## v13 変更点

- v12で「スタート地点から動けない」「ダンジョンが表示されない」と見えた問題への暫定対応。
- 初期位置を、起動直後から壁・通路・配置物が視認できる確認用通路へ変更。
- `初期位置` ボタンも同じ確認用位置へ戻るよう変更。
- 3D描画、パーティ表、罠検知、宝箱イベントの基本仕様はv12を維持。
- CSS/JSのキャッシュバスターを `20260609-v13` に更新。
