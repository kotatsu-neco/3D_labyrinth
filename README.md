# 灰の目録 3Dダンジョン試作 v04

本プロトタイプは、ゲーム全体ではなく「固定グリッド式のWebGLポリゴン3Dダンジョン表示」だけを検証するための試作です。

## v04 の変更点

- 扉の表示を修正
  - 扉の左右枠・上枠・扉板をセル中心基準で対称配置
  - 扉が少しズレて見える問題の修正を意図
- 扉を開いた後の表示を修正
  - 開扉後も扉枠を残す
  - 開いた扉板を通路脇に残す
  - 「扉を開くと何もなくなる」状態を廃止
- 簡易マップ表示を更新
  - 閉扉は `D`
  - 開扉は `O`
- キャッシュバスターを `20260609-v04` に更新

## 操作

- ↑ / W: 前進
- ↓ / S: 後退
- ← / A: 左旋回
- → / D: 右旋回
- 調べる / Enter / Space: 前方または現在地を調べる
- 初期位置: 開始地点に戻る
- 簡易マップ: デバッグ用マップ表示切替

## 確認のしかた

`index.html` をブラウザで開いてください。
GitHub Pages等に置いた場合も、外部ライブラリなしで動作します。

HTML本体のキャッシュが残る場合は、ページURL末尾に `?v=20260609-v04` を付けて開いてください。

## 注意

- これはダンジョン基盤の検証用です。
- UI、戦闘、育成、鑑定、目録はまだ本実装していません。
- 壁・床・天井テクスチャは外部画像ファイルではなく、起動時にCanvasで生成してWebGLテクスチャとして貼っています。
- iPhone Safari実機での見え方は未確認です。
