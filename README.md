# タイトル未定 地下方舟3Dダンジョン試作 v23f

タイトルは未確定です。v23fでは、iPhone Safari実機確認で指摘された戦闘画面のデッドスペース、右利き操作のしにくさ、閉じられないLOG小窓、ステータス詳細画面の見づらさを修正対象にしました。

## 実行方法

`index.html` をブラウザで開きます。

GitHub Pages等で配布する場合は、キャッシュ更新のため `style.css` と `app.js` に `20260611-v23f` のクエリを付けています。

## v23f 変更点

- `BUILD_VERSION` を `v23f` に更新。
- `index.html` のタイトル、HUD表示、キャッシュバスターを `v23f` に更新。
- 戦闘画面の入力エリアを下部2列の操作パッドへ変更。
- 戦闘コマンドを以下の固定配置に整理。
  - 攻撃 / 守る
  - 呪文 / 道具
  - 逃げる / 戻る
- 呪文・道具が使えない場合もボタン位置が崩れないよう、無効ボタンとして表示。
- 対象選択・呪文選択・道具選択の戻るボタンを、下部操作パッド内の大きいボタンとして配置。
- LOG小窓に閉じるボタンを追加。
- LOGを閉じた後も `LOG` ボタンから再表示できるように変更。
- LOG小窓を画面中央下ではなく、敵表示エリア右側の空き領域寄りに配置。
- ステータス詳細画面を左: 基本情報・能力値、右: 所持アイテムの2カラムへ変更。
- `ALIGN / RACE / CLASS` を縦並びに変更。
- `HITS / AC / STATUS / GOLD` を縦並びに変更。
- `ATTRIBUTES` ラベルを削除し、能力値を縦並びに変更。
- 所持アイテム欄を能力値欄の右側に配置。
- アイテムの `quantity` / `weight` は導入していません。

## v23fで変更していないこと

- 文字送り速度は `18ms` のまま。
- 敵画像はv21の低解像度・減色版を引き続き使用。
- 敵HP生成とプレイヤー攻撃によるHP減少の暫定処理は維持。
- キャラクター別コマンド入力、行動キュー、対象選択、呪文選択、道具選択の骨格はv23eの前提を維持。
