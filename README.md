# タイトル未定 地下方舟3Dダンジョン試作 v23e

タイトルは未確定です。v23eでは、戦闘画面をキャラクター別コマンド入力前提で再構築し、LOGを通常レイアウトから外して浮動小窓として扱う骨格へ変更しました。

## 実行方法

`index.html` をブラウザで開きます。

GitHub Pages等で配布する場合は、キャッシュ更新のため `style.css` と `app.js` に `20260610-v23e` のクエリを付けています。

## v23e 変更点

- `docs/BATTLE_SCREEN_DESIGN_RULES.md` を同梱。
- 戦闘画面を、キャラクター別コマンド入力前提の構造へ変更。
- 戦闘画面の通常フローを以下の3領域に整理。
  - 敵表示
  - パーティ状態
  - 入力エリア
- LOGを通常フロー内の1段から外し、戦闘ウィンドウ内の浮動LOG小窓に変更。
- 入力エリアに以下の状態を追加。
  - キャラクター別コマンド選択
  - 対象選択
  - 呪文選択
  - 道具選択
  - 全員入力後の決定/戻る
- 行動キューを追加。
- 戻る操作で前キャラクターの入力へ戻る骨格を追加。
- パーティ状態に、現在入力中/入力済み/未入力を示す記号を追加。
- `BUILD_VERSION` とキャッシュバスターを `v23e` に更新。
- `REPORT.md` / `docs/reports/REPORT_v23e.md` を追加。

## v23eで変更していないこと

- 文字送り速度は `18ms` のまま。
- 敵画像はv21の低解像度・減色版を引き続き使用。
- 敵HP生成とプレイヤー攻撃によるHP減少の暫定処理は維持。
- キャラクターステータス画面のv23c調整は維持。
