# 灰の目録 3Dダンジョン試作 v01

## 内容

「擬似3D」ではなく、WebGLで壁・床・天井をポリゴンとして描く、固定グリッド式3Dダンジョンの最小試作です。

## 起動方法

1. zipを展開します。
2. `index.html` をブラウザで開きます。
3. iPhoneで試す場合は、展開したフォルダをローカルサーバーで配信してアクセスするのが安定です。

外部ライブラリは使っていません。Three.js等のCDNも不要です。

## 操作

- ↑: 前進
- ↓: 後退
- ←: 左を向く
- →: 右を向く
- 調べる: 前方の壁・扉・イベントを調べる
- 簡易マップ: デバッグ用の簡易マップ表示を切り替え

キーボード操作も対応しています。

- W / ↑: 前進
- S / ↓: 後退
- A / ←: 左旋回
- D / →: 右旋回
- Space / Enter: 調べる

## 実装済み

- WebGLによる実ポリゴン3D描画
- 固定グリッド式の位置・方角管理
- 90度旋回
- 1マス移動
- 壁による移動阻止
- 閉じた扉の表示
- 調べる操作で扉を開く
- 階段マーカー表示
- イベントマーカー表示
- スマホ縦画面用UI
- 前衛3人・後衛3人のパーティ枠
- デバッグ用簡易マップ
