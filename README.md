# タイトル未定 地下方舟3Dダンジョン試作 v23g

タイトルは未確定です。v23gでは、v23fのiPhone Safari実機確認結果を受けて、ステータス詳細画面の所持アイテム上寄せ、AGE表示追加、戦闘画面のコマンド領域・LOG小窓の拡大を行いました。

## 実行方法

`index.html` をブラウザで開きます。

GitHub Pages等で配布する場合は、キャッシュ更新のため `style.css` と `app.js` に `20260612-v23g` のクエリを付けています。

## v23g 変更点

- `BUILD_VERSION` を `v23g` に更新。
- `index.html` のタイトル、HUD表示、キャッシュバスターを `v23g` / `20260612-v23g` に更新。
- ステータス詳細画面の所持アイテム欄を均等割付けではなく上寄せ表示に変更。
- ステータス詳細画面に `AGE` を追加。
- パーティメンバーのプロトタイプ表示データに `age` を追加。
- 戦闘画面の敵表示欄で、敵カードが縦に伸びすぎないよう2列配置寄りに調整。
- 戦闘画面の下部コマンドパッドを拡大。
- 戦闘LOG小窓を拡大し、本文表示領域を増加。
- LOG小窓の閉じる/再表示構造はv23fの方針を維持。
- アイテムの `quantity` / `weight` は導入していません。

## v23gで変更していないこと

- 戦闘画面はキャラクター別コマンド入力のまま。
- 対象選択・呪文選択・道具選択・戻るの状態分離は維持。
- 戦闘LOGは通常レイアウト内の段ではなく、独立した浮動小窓のまま。
- 敵画像はv21の低解像度・減色版を引き続き使用。
- 敵HP生成とプレイヤー攻撃によるHP減少の暫定処理は維持。
