# 灰の目録 3Dダンジョン試作 v02

本プロトタイプは、ゲーム全体ではなく「固定グリッド式のWebGLポリゴン3Dダンジョン表示」だけを検証するための試作です。

## v02 の変更点

- 一歩移動のアニメーションを調整
  - 160msの単純移動から、240msの加減速 + 軽い上下動 + 着地感へ変更
  - 壁衝突時に短いバンプ動作を追加
- 左右旋回を調整
  - 130msから190msへ変更
  - 加減速と軽い横揺れを追加
- 通路幅を拡張
  - 描画上の1マス幅を v01 の 1.0 から 1.5 に変更
  - 内部グリッド座標・当たり判定は維持
- 壁・床・天井の視認性を改善
  - WebGL上で生成した石壁テクスチャを壁面に貼付
  - 床と天井にも簡易テクスチャを適用
  - 霧を弱め、近距離の壁面が黒く潰れすぎないよう調整

## 操作

- ↑ / W: 前進
- ↓ / S: 後退
- ← / A: 左旋回
- → / D: 右旋回
- 調べる / Enter / Space: 前方または現在地を調べる
- 初期位置: 開始地点に戻る
- 簡易マップ: デバッグ用マップ表示切替

## 確認のしかた

`index.html` をブラウザで開いてください。
GitHub Pages等に置いた場合も、外部ライブラリなしで動作します。

## 注意

- これはダンジョン基盤の検証用です。
- UI、戦闘、育成、鑑定、目録はまだ本実装していません。
- 壁テクスチャは外部画像ファイルではなく、起動時にCanvasで生成してWebGLテクスチャとして貼っています。
