# 灰の目録 3Dダンジョン試作 v06

WebGLで描画する、固定グリッド式の軽量ポリゴン3Dダンジョン試作です。

## 起動方法

`index.html` をブラウザで開いてください。
GitHub Pages等に配置する場合は、フォルダ内のファイル一式をそのまま配置します。

## 操作

- ↑: 前進
- ↓: 後退
- ←: 左旋回
- →: 右旋回
- 調べる: 前方または現在地の配置物を調べる / 閉じた扉を開く / 壁レバーを操作する
- 初期位置: 開始位置に戻る
- 簡易マップ: デバッグ用マップ表示

## v06 変更点

- 宝箱と祭壇を通路中央の障害物ではなく、壁際の非ブロック配置物に変更。
- 宝箱と祭壇のサイズを小型化し、通路移動を阻害しないように変更。
- 扉のプロシージャル縞・金属帯表現を廃止し、まだらに見えにくい単色寄りの扉に変更。
- レバー操作時に対応扉 `x5 y4` を開閉するように変更。
- レバーのメッセージを「対応する扉が開く/閉じる」に変更。
- 現在地にある宝箱・祭壇も「調べる」で反応するように変更。
- CSS/JSのキャッシュバスターを `20260609-v06` に更新。
